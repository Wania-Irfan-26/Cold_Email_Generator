#  AI Cold Email Agent (Streamlit App)

An intelligent multi-agent AI system that researches companies and generates highly personalized cold emails automatically.

This project automates the outreach process using AI agents that search, scrape, analyze, and generate targeted emails based on real company data.

---

##  Project Overview

This system streamlines cold outreach by:

- Researching a company using search tools  
- Scraping relevant website information  
- Analyzing business context  
- Generating personalized cold emails  
- Delivering structured, high-quality output  

Built as a scalable AI-powered automation solution.

---

##  System Architecture

The application follows a multi-agent workflow:

### 1. Research Agent
- Uses Serper API for web search  
- Identifies company news, products, and activity  

### 2. Scraping Agent
- Extracts important website content  
- Collects structured business information  

### 3. Strategy Agent
- Finds personalization angles  
- Matches company profile with outreach strategy  

### 4. Email Writer Agent
- Generates high-converting cold email copy  
- Creates subject lines and strong call-to-action
